Hexo安装

打开命令行或者终端输入: $ npm install -g hexo-cli

然后Enter（回车键）

等待终端显示完全安装后再进 go on

初始化Hexo

选择一个文件夹来存放所有hexo文件

如果你跟我一样用MAC，建议在新建一个文件夹，然后右键文件夹显示简介

复制下来蓝色那段文字然后加上当前文件夹名称，比如上图我复制出来就是 /Users/levenls ，然后加上文件夹名就是 /Users/levenls/Develop

打开终端 输入 cd /Users/levenls/Develop 就进入到当前文件夹

以后每次对hexo进行操作都请进入到当前文件夹

按照顺序执行：

  • $ hexo init blog
  • $ cd blog
  • $ npm install

然后，你就会得到如下图的文件结构

生成网站

$ hexo generate或者$ hexo g

生成完之后

$ hexo server或者$ hexo s

在浏览器输入

http://localhost:4000
